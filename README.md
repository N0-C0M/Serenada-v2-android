# Serenada v2
A simple, privacy-focused 1:1 video calling application built with WebRTC. No accounts, no tracking, just instant video calls.
## Backend Repository
To fully utilize this application, you need the server-side component:
 https://github.com/N0-C0M/serenada-v2
## Configuration & Setup
Before building the application, you must configure the connection to your server.
Changing the API URL
 * Open the project in Android Studio.
 * Navigate to the following path:
   app > java > com.example.serenada_v2 > MainActivity
 * Locate the variable:
   private final String URL = "http://current-url.com";

 * Replace the value with your own server URL.
## Building the Project
 * Launch Android Studio.
 * Select File > Open and choose the serenada_v2 folder.
 * Wait for the Gradle sync to finish.
 * Connect your Android device or start an emulator.
 * Click the Run button (Green Arrow) or press Shift + F10.
